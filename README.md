## :tada: 安装

使用`pnpm`安装依赖

```shell
pnpm install
```

构建可视化组件库

```shell
pnpm build:components
```

运行

```shell
pnpm dev
```

组件开发

提供了脚手架快速搭建组件文件目录模板

安装脚手架

```shell
pnpm build:cli
```

生成模板

```shell
pnpm gen:wd
```
